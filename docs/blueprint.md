# **App Name**: QR Canvas

## Core Features:

- Dynamic QR Code Generation: Generate QR codes in real-time based on user input, supporting various data types like URL, Text, Phone, Email, and WiFi.
- Live QR Code Preview: Display a large, interactive preview of the QR code as customization options are adjusted, ensuring a seamless user experience.
- QR Code Customization Panel: Allow users to customize QR code foreground/background colors, upload a logo (with preview), select output size, and set error correction levels (Low, Medium, High, Highest).
- Multi-format Download Options: Enable downloading the generated QR code in popular image formats: PNG, SVG, and JPG.
- Content Type Tabs: Provide tabbed navigation for different QR code content types (URL, Text, Phone, Email, WiFi) to simplify input.
- Responsive UI Design: Ensure the application adapts gracefully across various device sizes, offering an optimal experience on both mobile and desktop.
- AI-Powered QR Content Refiner: An AI tool that can suggest more concise or impactful wording for the text embedded in the QR code, enhancing clarity or call-to-action effectiveness.

## Style Guidelines:

- Dark Color Scheme: Utilizing a very dark charcoal with a subtle green tint as the background (RGB: #1B2B1F), providing a modern and sophisticated foundation.
- Primary Color: A vibrant, clear green (RGB: #26EA56) to highlight interactive elements, calls to action, and key information, standing out against the dark background.
- Accent Color: A secondary, lighter yellow-green (RGB: #ADF159) used for subtle accents, selected to be analogous to the primary and maintain a harmonious yet distinct visual language.
- Headline and body font: 'Inter' (sans-serif) for its clean, modern, and highly legible appearance, ensuring excellent readability across all screen sizes.
- Use minimalist, outline-style icons with a modern aesthetic, primarily in shades that contrast well with the dark theme and align with the primary green accent for consistency.
- A responsive, mobile-first layout with generous spacing. On desktop, a clear two-column structure separates input forms from the large, prominent QR code preview. Components should be centered for optimal user focus.
- Subtle and swift animations for interactive elements, such as input focus states, tab transitions, and the QR code generation process, to enhance the perception of speed and responsiveness.
